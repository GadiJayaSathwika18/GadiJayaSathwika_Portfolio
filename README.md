- ⚙️ Easy customization through `portfolio.js`

## 🔧 Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Git

### Installation

```bash
# Clone the repo
git clone https://github.com/GadiJayaSathwika18/GadiJayaSathwika_Portfolio.git
cd GadiJayaSathwika_Portfolio

# Install dependencies
npm install

# Start the development server
npm start
